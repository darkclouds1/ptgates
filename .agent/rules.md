# User Rules for Antigravity

These rules must be followed in every session for this project.

## 1. Operating System Commands

- **OS Environment**: `Windows`
- **Search Command**: NEVER use `grep`. ALWAYS use `findstr`.
  - _Reason_: The environment is Windows PowerShell/CMD where `grep` is not available by default.

## 2. Communication Language

- **Language**: Korean (한국어)
- **Requirement**: All communication with the user, including explanations, summaries, and notifications, MUST be in Korean.

깔끔하고 모던한 문의 및 제안 이메일 폼 플러그인

== Description ==

Contact Email Form은 방문자가 쉽게 문의 및 제안을 이메일로 보낼 수 있는 플러그인입니다.

주요 기능:
* 깔끔하고 모던한 디자인
* AJAX 기반으로 페이지 새로고침 없이 작동
* Hostinger SMTP를 통한 안정적인 이메일 발송
* 반응형 디자인 (모바일 최적화)
* 간단한 숏코드 사용

== Installation ==

1. 플러그인을 업로드하고 활성화합니다.
2. 원하는 페이지나 포스트에 [contact_email_form] 숏코드를 추가합니다.
3. Elementor를 사용하는 경우, 숏코드 위젯에 [contact_email_form]을 입력합니다.

== 사용 방법 ==

숏코드: [contact_email_form]

이 숏코드를 페이지, 포스트, 또는 위젯 영역에 추가하면 문의 폼이 표시됩니다.
